# PKTWinNode

A Windows desktop application for deploying and managing PKT CJDNS node using WSL2.

## Overview

PKTWinNode simplifies running a PKT CJDNS node on Windows by automating the deployment and configuration of an Ubuntu WSL2 distribution.

## Requirements

### System Requirements
- Windows 10/11 Pro, Enterprise, or Education (for Hyper-V)
- x64 processor with virtualization support
- 4GB RAM (8GB recommended)
- 10GB free disk space
